# VapeStore - Website de Venda de Vapes e Pods

## Descrição do Projeto
Plataforma completa de e-commerce especializada na venda de vapes, pods, acessórios e e-liquids.

## Tecnologias Utilizadas
- Frontend: React.js
- Backend: Node.js com Express
- Banco de Dados: MongoDB
- Estilização: Tailwind CSS
- Pagamentos: Stripe
- Autenticação: JWT

## Funcionalidades Principais
- Catálogo de produtos
- Carrinho de compras
- Sistema de autenticação de usuários
- Processo de checkout
- Área administrativa para gestão de produtos
- Responsividade mobile

## Pré-requisitos
- Node.js (v16+)
- MongoDB
- npm ou yarn

## Instalação

### Clonar Repositório
```bash
git clone https://github.com/seu-usuario/vapestore.git
cd vapestore
```

### Instalação de Dependências
```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

## Configuração
1. Criar arquivo `.env` com variáveis de ambiente
2. Configurar credenciais de banco de dados
3. Definir chaves de API

## Executando o Projeto
```bash
# Iniciar backend
npm run server

# Iniciar frontend
npm run start
```
